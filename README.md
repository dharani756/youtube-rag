Overview

A Retrieval-Augmented Generation (RAG) application that allows users to ask questions about any YouTube video.

The system extracts video transcripts, converts them into embeddings, stores them in a FAISS vector database, retrieves relevant context, and generates answers using an LLM.
Features
Extract transcripts from YouTube videos
Convert transcripts into vector embeddings
Store embeddings using FAISS
Semantic search over video content
Ask questions about video content
AI-generated answers using Gemini
React frontend
FastAPI backend
